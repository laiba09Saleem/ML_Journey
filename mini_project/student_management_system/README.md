# Student Management System

A simple, industry-ready Student Management System implemented in Python. This project demonstrates basic CRUD (Create, Read, Update, Delete) operations for managing student records, with data persistence using CSV and JSON formats.

## Features
- Add, view, update, and delete student records
- Data storage in both CSV and JSON files
- Command-line interface for user interaction
- Modular and maintainable code structure

## File Structure
```
student_management_system.py   # Main application script
students.csv                  # Student data in CSV format
students.json                 # Student data in JSON format
```

## Getting Started

### Prerequisites
- Python 3.7 or higher

### Installation
1. Clone the repository or download the project files.
2. Ensure Python is installed on your system.

### Usage
Run the main script using the command line:
```bash
python student_management_system.py
```
Follow the on-screen prompts to manage student records.

## Code Overview
- **student_management_system.py**: Contains all logic for interacting with student data, including reading/writing to CSV and JSON, and the CLI interface.
- **students.csv**: Stores student records in CSV format.
- **students.json**: Stores student records in JSON format.
